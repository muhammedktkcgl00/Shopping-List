🛒 Alışveriş Listesi Uygulaması

Basit ve kullanışlı bir alışveriş listesi uygulaması.
HTML, CSS ve JavaScript kullanılarak geliştirilmiştir.

🚀 Özellikler

✅ Yeni ürün ekleme

🗑️ Ürün silme

⚠️ Boş ürün eklemeyi engelleme

🔄 Anlık liste güncelleme (DOM render mantığı)

🛠️ Kullanılan Teknolojiler

HTML5

CSS3

Vanilla JavaScript

📌 Proje Amacı

Bu proje, JavaScript temel konularını pekiştirmek amacıyla geliştirilmiştir.
